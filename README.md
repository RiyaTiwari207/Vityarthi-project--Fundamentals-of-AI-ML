Overview

ReviewShield is an AI-powered system designed to detect fake product reviews using Machine Learning and Natural Language Processing (NLP). The system analyzes user reviews and classifies them as REAL or FAKE, helping users make better decisions while shopping online.

Objectives

To identify fake product reviews using AI techniques

To apply NLP for text processing

To use Machine Learning for classification

To improve trust in online platforms

Features

 Detects fake and real reviews
 
 Uses Machine Learning (Logistic Regression)
 
 Applies NLP techniques for text preprocessing
 
 Combines rule-based logic with ML

 Simple user interface with quick results

 Technologies Used

Programming Language: Python

Libraries: pandas, scikit-learn, re

Concepts Used: Machine Learning, NLP, Rule-Based System

System Working

User enters a product review and rating

Text is cleaned using NLP preprocessing

TF-IDF converts text into numerical format

Machine Learning model predicts authenticity

Rule-based logic checks suspicious patterns

Final output is displayed as:

 REAL

 FAKE
